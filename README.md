# How to run
  using maven
  ```bash
  mvn spring-boot:run
  ```

# UI for glass view
  - H2
    - url:
      ```
      http://localhost:9090/console/h2-ui
      ```
    - username to login: `admin`
    - password: `password`

  - Swagger url
    ```
    http://localhost:9090/console/swagger-ui
    ```
